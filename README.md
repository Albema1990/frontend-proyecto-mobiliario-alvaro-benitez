# Frontend Catálogo Mobiliario 

Proyecto final del Bootcamp Neoland Web Development.

## Descripción

Aplicación web para gestionar un catálogo de productos de mobiliario.

## Tecnologías

- React
- Vite
- CSS
- JavaScript

## Como ejecutar el proyecto

```shell
npm install
npm run dev
```
