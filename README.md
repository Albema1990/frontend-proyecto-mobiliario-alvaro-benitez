# Frontend Catálogo Mobiliario 

Proyecto final del Bootcamp Neoland Web Development.

## Descripción

Aplicación web para gestionar un catálogo de productos de mobiliario.

## Tecnologías

- React
- Vite
- CSS
- JavaScript

## Clonar el repositorio

```
git clone https://github.com/Albema1990/frontend-proyecto-mobiliario-alvaro-benitez.git

git switch dev
```

## Como ejecutar el proyecto

```shell
npm install
npm run dev
```
